# Favly App

## About
An app utilizing the Giphy Search API.<br>
Allows you to search by keyword, trending or random.<br>
You can save and store your favorite gifs locally.

## Tech
- Vite
- TypeScript
- React
- React Router
- Tailwind CSS
- Zustand
- GSAP
